# Customer Churn Prediction (Logistic Regression + Feature Engineering)

This project predicts whether a customer is likely to churn based on demographic, account, and activity data. It includes a full ML pipeline with data cleaning, preprocessing, feature engineering, model training, and evaluation.

## Key Features:
- Handles categorical and numerical features
- Feature scaling and encoding
- Logistic Regression classifier
- Evaluation using accuracy, ROC-AUC, confusion matrix, and classification report
- Predictive pipeline for new customers

## Tech Stack:
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

## Dataset:
Includes customer demographics, tenure, balance, product holdings, credit card usage, and activity level.
